##  立执行函数

一般情况下，函数是先定义，然后被调用执行。而立执行函数可以在定义之后立即执行。

将函数的定义用圆括号括起来，再在后面加上一个圆括号，就是一个立执行函数。如下是一个立执行匿名函数的示例：

```js
(function() {
        console.log('hello')
})()
```

也可以写成箭头函数：

```js
(() => {
        console.log('hello')
})()
```

立执行函数可以定义和传入参数：

```js
((a, b) => {
        console.log(a + b)
})(1, 2)
```
